Automating Daily Office Tasks
with Google Apps Script

Boost productivity in Google Workspace with automation

Abstract

This article explores the use of Google Apps Script to automate common office tasks like sending reports, organizing emails, and managing calendar events. With minimal code, professionals can enhance productivity within the Google Workspace ecosystem.

1. Introduction

Office work often includes repetitive tasks that consume valuable time. Google Apps Script offers a JavaScript-based solution to automate tasks inside Gmail, Sheets, Docs, and Calendar. This blog demonstrates practical use cases to streamline daily routines.

2. Related Work

Alternatives like Zapier and Microsoft Power Automate offer no-code automation, but Google Apps Script provides greater flexibility and native integration with Workspace tools.

3. Approach

We identify common office workflows and implement small scripts to automate them. Key criteria include frequency, time saved, and integration depth with Google Workspace.

4. Examples in Daily Office

4.1. Send Daily Email Report

function sendReport() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Summary");
  const data = sheet.getRange("A1:B10").getValues();
  let body = "Weekly Summary:\\n\\n";
  data.forEach(row => body += row.join(": ") + "\\n");
  MailApp.sendEmail("manager@example.com", "Weekly Report", body);
}

4.2. Archive Old Emails

function archiveOldEmails() {
  const threads = GmailApp.search('label:Promotions older_than:14d');
  threads.forEach(t => t.moveToArchive());
}

4.3. Calendar Event from Sheet

function scheduleMeeting() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Meetings");
  const [title, date, time] = sheet.getRange("A2:C2").getValues()[0];
  const calendar = CalendarApp.getDefaultCalendar();
  const startTime = new Date(`${date} ${time}`);
  calendar.createEvent(title, startTime, new Date(startTime.getTime() + 60 * 60 * 1000));
}

4.4. Auto Reply to Form Submission

function onFormSubmit(e) {
  const name = e.namedValues["Name"][0];
  const email = e.namedValues["Email"][0];
  MailApp.sendEmail(email, "Thank You!", `Hi ${name},\\n\\nThanks for your submission!`);
}
